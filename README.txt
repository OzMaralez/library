Инструкция по запуску приложения:

Шаг 1:
    - Переименовать файл .env.sample в .env
    - Указать в переименованном файле имя пользователя БД (DB_USER) и пароль (DB_PASSWORD)

Шаг 2:
    - В терминале перейти в папку с проектом
    - Запустить команду docker compose build
    - Далее команду docker compose up
